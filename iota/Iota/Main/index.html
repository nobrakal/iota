Provided a configuration and a Lexbuf to parse, this function compiles a Iota code. It:

  1. Parses it using Menhir.
  2. Typechecks it (with Typecheck).
  3. Compiles it to a much simpler final object (with Final).
  4. Tries to infer some information (with Guard_inference)
  5. Ensures its validty (with Structure).
